Detailed Description

Inmemory-Template Class

This is an template that is not loaded everytime you run it, but it's content is stored in-memory before you run it the first time. This is good for small templates you need often (for example a line of a table). IM-templates don't support some advanced features real templates as they don't support inline-code without using eval.

Author
Alexander Kastius

Member Function Documentation

run ( )

Use this to run the InMemoryTemplate, important: It will search for the inmemory template, if its not included yet.
