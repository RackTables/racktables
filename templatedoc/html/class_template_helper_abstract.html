Detailed Description

Use this to easily create your own template helpers, just extend the class and implement the generate($params) function. This function should echo the content using the passed parameters, thoose will be already parsed (placeholder will be replaced with values, or with "" when the placeholder didn't exist). You should make shure that your helper will output nothing, not even the surrounding tags, when launched with empty parameters.
