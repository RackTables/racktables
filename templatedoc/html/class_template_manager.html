generatePseudoSubmodule (   $placeholder,
TemplateModule  $parent = null,
  $output = array() 
)

Generates a 'pseudo'-submodule. It creates a TemplatePseudo instead of a TemplateInMemory/TemplateModule The difference between a TemplateModule and TemplatePseudo is that TemplatePseudo doesn't actually use a template file. Instead it stores the content of its local placeholder array in its parent module.

This allows it to use TemplatePseudo to create loop-arrays without using actual arrays in your code. To do so, you just create multiple pseudo-submodules on one placeholder, you can access the content of thoose modules in a loop.
